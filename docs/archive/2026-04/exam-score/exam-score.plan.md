# Plan: 기출문제 점수 및 합격 표시

> 기출문제 회차를 100% 풀었을 때 점수, 틀린 개수, 합격/불합격 여부를 표시하는 기능

## 배경

현재 기출문제 탭에서 연도별 회차를 선택하면 학습 모드 선택 모달이 표시된다.
모든 문제를 다 풀면(100%) "전체 다시 풀기" 버튼만 나타나는데,
**점수가 몇 점인지, 몇 개 틀렸는지, 합격인지 불합격인지** 알 수 없는 문제가 있다.

정보처리기사 실기는 20문제 각 5점 배점, 60점 이상 합격이므로
이 기준에 맞는 결과 표시가 필요하다.

## 요구사항

### FR-01: 점수 계산 및 표시
- 100% 완료된 기출 회차의 모달에 점수 표시
- 점수 = (정답 수 / 전체 문제 수) × 100
- 맞은 개수, 틀린 개수 표시

### FR-02: 합격/불합격 판정
- 60점 이상: 합격 표시 (녹색 배지)
- 60점 미만: 불합격 표시 (빨간색 배지)

### FR-03: 모달 UI 개선
- 100% 완료 시 모달 상단에 점수 결과 영역 추가
- 기존 "전체/학습완료/미학습" 통계 영역을 점수 결과로 대체
- "전체 다시 풀기" 버튼은 유지

### FR-04: 틀린 문제만 다시 풀기
- 100% 완료 시 틀린 문제가 있으면 "틀린 문제만 다시 풀기" 버튼 추가
- 해당 회차의 incorrect 상태 문제만 필터링하여 퀴즈 시작
- 틀린 문제가 0개이면 버튼 미표시

### 데이터 흐름
- 이미 존재하는 `CategoryStats`의 `correctCount`, `incorrectCount`, `totalQuestions` 활용
- `useUserStore.getCategoryStats()`로 점수 계산 가능 (추가 스토어 변경 불필요)

## 점수 계산 로직

```
score = Math.round((correctCount / totalQuestions) * 100)
isPassed = score >= 60
```
