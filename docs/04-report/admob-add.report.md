### 3.2 세부 구현 내용

#### 광고 시청 시나리오

| 시나리오 | 조건 | 동작 |
|---------|------|------|
| 처음 도전 | `seenCount === 0 && !quizCanResume` | 광고 시청 후 진입 |
| 전체 다시 풀기 | `mode === 'all'` | 광고 시청 후 진입 |
| 이어서 풀기 | `quizCanResume === true` | 광고 없이 즉시 진입 |
| 틀린 문제 풀기 | `mode === 'incorrect'` | 광고 없이 즉시 진입 |
| 진행 중 재진입 | 이전 진행 상태 | 광고 없이 즉시 진입 |

#### 광고 로드 흐름

```
광고 준비 (useRewardedAd)
  ↓
광고 표시 필요 판단
  ├─ Expo Go 환경: 노옵 (개발 편의)
  ├─ 광고 준비됨: 표시
  └─ 광고 미준비: 재로드
  ↓
사용자 광고 시청
  ├─ 완료: EARNED_REWARD 이벤트 발생
  ├─ X 클릭: 광고 취소 (보상 없음)
  └─ 오류: 재시도
  ↓
광고 종료 후 자동 재로드
```

## 4. 기술 스택 및 구현 세부사항

### 4.1 사용된 기술
- **플랫폼**: Expo (React Native)
- **타입스크립트**: Strict 모드 준수
- **상태 관리**: Zustand (기존)
- **광고 라이브러리**: `react-native-google-mobile-ads`

### 4.2 주요 함수/훅

#### `useRewardedAd()` 훅 시그니처
```typescript
interface UseRewardedAdReturn {
  isAdLoaded: boolean
  showAd: () => Promise<void>
  loadAd: () => void
}

const useRewardedAd = (onReward: () => void): UseRewardedAdReturn
```

#### 광고 표시 조건 로직
```typescript
// exam.tsx에서
const shouldShowAd = 
  (seenCount === 0 && !quizCanResume) || 
  mode === 'all'

if (shouldShowAd) {
  await showAd()
}
```
